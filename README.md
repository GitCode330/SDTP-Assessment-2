# MAL2021 Assessment 2: Software Development Tools and Practices

## LMS Application with Spring Boot Backend and Swing GUI

### Features
- REST API endpoints for student enrollment management
- Java Swing desktop client
- Comprehensive unit and integration tests
- Usability testing with GUI improvements

### Technologies
- Java 17
- Spring Boot 3.1.5
- JUnit 5
- Mockito
- Java Swing

### How to Run
1. Run Spring Boot application: `SoftwareTestingApplication.java`
2. Run Swing GUI: `StudentEnrollmentGUI.java`

### Endpoints
- `GET /api/students/{id}/enrollments` - Get student enrollments
- `GET /api/students/active` - List active students
- `GET /api/instructors/most-active` - Most active instructor
- `GET /api/instructors/inactive` - Instructors with no enrollments
